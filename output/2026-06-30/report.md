### 1. WorldQuant 资金流 Alpha 构造逻辑

**核心思想**  
资金主动买卖的强度与方向，在横截面和时间序列上对价格形成非线性影响，尤其关注大资金行动后的短期动量衰竭与反转。

**公式逻辑（可操作）**  
- **大单净主动买入比率与未来1日反转**：  
  将分钟线主动买卖单按成交额前20%定义为大单，计算当日大单净主动买入额占成交额比（`big_order_net_pct`）。做次日收益率反向排序，即 `big_order_net_pct` 极高的股票次日容易回落，极低的次日容易反弹。  
- **尾盘30分钟资金流入强度衰减**：  
  计算最后30分钟净主动流入金额相对于全天净主动流入的占比尾盘加速流入信号。若前3.5小时资金平稳但尾盘急剧涌入，未来2日超额收益为负（诱多）。做多尾盘占比低、做空尾盘占比高的股票。  
- **资金流与隔夜跳空交互**：  
  `overnight_gap * money_flow_20d_zscore`，当20日资金流处于高位且出现向上跳空时，视为短期过度反应，未来1日反转下跌。交互项系数为负。

**期望 IC 范围（A股实证）**  
日频 Rank IC 通常 **0.02–0.04**（绝对值），正交化规模/行业后可能接近0.03。尾盘衰退信号 IC 中位数约 −0.025–−0.035。

**可翻译成的算子约束**  
- 时序算子：`ts_delta(amount_net_active, 1)` / `ts_mean(turnover, 20)`  
- 横截面分位数：`rank(big_order_net_pct)` 与 `shift(return, -1)` 做斯皮尔曼相关  
- 条件交互：`if(last30_flow_ratio > 0.7 and morning_flow_pct < 0.3, signal = -1)` 的组合编码  
- 衰减表达：`ts_decay_linear(last30_flow_pct, 3)` 对近3日尾盘信号做线性衰减加权

### 2. 分钟级资金流（主力/散户/大单）的有效组合方式

**核心思想**  
主力与散户行为的背离程度，是短期定价错误的先行指标；大单持续性与日内集中度的组合比单因子稳定得多。

**公式逻辑**  
- **主力-散户净买入背离强度**：  
  按单笔成交额划分主力（>20万元/单）和散户（<4万元/单）。计算过去5日主力累计净买入额 Z-score 与散户累计净买入额 Z-score 之差（`smart_dumb_divergence`）。背离越极端（主力买、散户卖，或反向），未来3日收益方向越偏向主力行为。横截面取多头为高背离正向端。  
- **大单成交集中度×持续性因子**：  
  定义每日大单净买强度 `big_net_intensity = big_net_amount / volume`。  
  持续性：`big_net_intensity > 0` 的连续天数。  
  集中度：日内大单成交金额的赫芬达尔指数（以分钟为组）。  
  组合：`rank(big_net_intensity_5d_persistence) * rank(intraday_big_hhi)`，选两者均高的股票。未来5日胜率提升，IC约0.03–0.05（中证800内）。  
- **早盘主力进场与全天市场情绪背离**：  
  若早盘30分钟主力净买极强，但指数同期下跌，则个股资金流“逆势”度越高，未来1日反弹越强。计算：`morning_big_net / morning_volume` 减去 `index_morning_return` 的 Z 值，做多最高组。

**期望 IC 范围**  
主力-散户背离因子的 5日IC 约 **0.025–0.04**；大单集中度持续性因子 IC 在 **0.03–0.05**，但换手率高，容量有限。早盘背离因子日IC 约 0.02–0.03。

**可翻译成的算子约束**  
- 分组：`ts_sum(if(amount>200000, net_buy, 0), 5) / ts_mean(turnover, 5)`  
- Z-score 差：`scale(main_z) - scale(retail_z)`  
- 持续性计数：`ts_count(big_net_intensity > 0, 5)`  
- 集中度：`ts_intra_herfindahl(big_amount_share, minute_bucket)`  
- 组合乘：`rank(persistence) * rank(hhi)` 视为合成算子

### 3. A股特有的资金流模式

**核心思想**  
A股存在涨跌停制度、北向资金联动、尾盘固定策略等独特模式，导致资金流在极端情形下具有高度可预测的反转或趋势特征。

**公式逻辑**  

**① 跌停板撬板资金流质量**  
- 当日从跌停板打开（或未封死）时，统计开板期间大单净买入额 / 全天成交额（`reversal_big_pct`）。  
- 若该比值 > 0.3，且开板后未再封跌停，则未来1–3日绝对收益显著为正。  
- 信号：选择 `reversal_big_pct > 0.3` 且 `close > open`，次日开盘买入持有2日。因子化表达为 `reversal_big_pct * (1 if not hit_limit_down_again else 0)`，横截面Rank IC 可达 0.05–0.08（事件驱动，不稳定）。  

**② 北向资金联动滞后**  
- 北向每日净流入额（沪深股通合计）在日内分钟频率上领先内资主力。  
- 构建因子：个股北向资金当日净买入额占成交额比（`north_net_ratio`），但滞后1日生效，因为信号在盘后才完全公布。  
- 做多北向资金连续3日净流入的股票（`ts_min(north_net_ratio, 3) > 0`），未来5–10日有稳定正IC，约 0.03–0.04（大盘股域更显著）。  

**③ 尾盘异动与ETF套利**  
- A股尾盘最后5分钟常出现显著放量且方向一致的异动（机构调仓或ETF再平衡）。  
- 定义尾盘异动因子：`tail_move = (close - price_t_minus_5) / price_t_minus_5` 与 `tail_volume / day_volume` 的乘积，然后做隔夜反转。即极端正异动次日低开，对应做空隔夜收益。  
- IC仅在隔夜收益（Open-to-Open）上有效，约 −0.015 至 −0.025。隔夜反转信号需要与波动率调整结合以降低噪声。

**期望 IC 范围**  
跌停板撬板因子为事件驱动，胜率>55%，Rank IC 均值 0.03–0.05 但覆盖率极低；北向滞后因子 IC 约 0.03–0.04（沪深300内）；尾盘异动反转 IC 较低，约 −0.01 到 −0.025，且需隔夜窗口。

**可翻译成的算子约束**  
- 跌停事件判定：`if(prior_close*0.9 ~ close)`，提取开板段大单净买：`ts_big_net_buy during limit_open_period`  
- 北向滞后：`shift(north_net_ratio, 1)` 与未来收益相关  
- 连续条件：`ts_all(north_net_ratio > 0, 3)`  
- 尾盘异动：`(close - ts_delay(close, 5)) / ts_delay(close, 5) * (last5_vol / day_vol)`  
- 隔夜反转：预测目标 `open_to_open_return` 并取负相关

## 🔄 下一轮建议

基于本轮方法论与发现的因子（最高IC=0.087，多为基础代数组合），下一轮改进需让因子更贴合资金流反转逻辑，并从“静态代数”向“时序+截面非线性”升级。具体建议如下：

---

### 1. 新增算子/变换及理由

**① 新增时序算子：`ts_delta`、`ts_rank`、`ts_mean`、`ts_max`/`ts_min`**  
- **理由**：方法论强调“短期动量衰竭与反转”“尾盘30分钟流入强度衰减”，这本质是时间序列上的相对变化和极值比较，单纯的`sub`、`div`无法刻画。例如，想要“尾盘30分钟净流入相对于全天净流入占比突然升高”，需要使用`ts_mean(flow, 30)/ts_mean(flow, 240)`或`ts_rank(flow, 30)`这类表达。引入时序算子才能让 GP 搜索到这些动态模式。

**② 新增截面算子：`rank`、`zscore`、`ind_neutralize`**  
- **理由**：资金流 Alpha 是做横截面排序（IC 导向），且 WorldQuant 逻辑中常对因子做行业市值中性化以避免风格暴露。当前因子表达式完全无截面比较，可能只是与自身绝对数值相关，IC 易受行业/市值干扰。加入截面排序和中性化算子可直接提升因子在横截面上的鲁棒性。

**③ 新增条件/分段算子：`if_else` 或 `where`**  
- **理由**：方法论明确提出“大单净主动买入比率极高则次日容易回落，极低则容易反弹”以及“尾盘占比低做多、高做空”，这本质是分段非线性效应（极高、极低分别对应反向，中等区间无效）。现有连续算子难以捕捉这种分界。加入条件算子后，GP 能进化出如 `if_else(X>threshold, -X, X)` 这类典型反转表达。

**④ 新增 `power`、`sign`、`max`/`min`**  
- **理由**：资金流与隔夜跳空的交互（如 `overnight_gap * mon` 截断处）可能涉及方向敏感性或乘方关系。`sign` 可提取方向，`max`/`min` 可捕捉“资金流与价格同向/反向时的不同效应”，提供更丰富的非线性。

---

### 2. 参数范围调整

**窗口大小**：  
- **理由**：方法论中的有效模式集中在 **30 分钟**（尾盘）、**1 日**（隔夜跳空）和 **2 日**（诱多后反应）。如果当前使用的都是日频数据，很可能完全丢失尾盘与全天的相对强度信息。  
- **调整**：若数据支持分钟级，应增加 **30 分钟、60 分钟、全天** 等多个窗口的时序特征；若仅日频，至少需要 **1 日、2 日、5 日** 的短窗口差分和均值算子，不能只靠无穷窗口。建议算子内置窗口列表：`[1, 2, 5, 10, 20]`（日频）或 `[30, 60, 120, 240]`（分钟频）。

**进化代数与种群规模**：  
- **理由**：当前最高 IC 仍未突破 0.09，表明搜索空间可能不够深或多样性不足，且新增算子后表达式复杂度将大幅上升。  
- **调整**：种群规模至少增大到 **500**（原可能100-200），代数提升至 **20~30 代**，同时启用 **锦标赛选择 + 自适应变异率** 以防止早熟。可在初期使用较大变异率（如 0.3），后期逐渐降低。

**过拟合控制参数**：  
- **建议**：在适应度函数中加入 **样本外 IC 衰减惩罚**（如要求 train IC 和 valid IC 差异不超过 0.03），直接过滤掉过于复杂的纯噪音组合。理由：资金流反转模式虽强，但极易过拟合特定日期，需加正则。

### 3. 被忽略的数据维度

- **理由**：方法论专门提到了尾盘、隔夜跳空，但当前因子完全未体现这些时段信息的拆分。原始特征表可能只有“日频净主动买入”等汇总指标，未将尾盘剥离。  
- **建议**：直接构造并输入 **“尾盘30分钟净主动买入占比”**、**“早盘60分钟净主动买入”**、**“隔夜跳空幅度”** 等衍生特征列，让 GP 拥有可直接组合的模块，
