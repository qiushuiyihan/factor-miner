### 1. WorldQuant式资金流Alpha：震荡市主动成交强度

**核心思想**  
主动资金在价格横盘期间的持续净流入，对上方阻力突破有预测作用。

**公式逻辑**  
计算每日主动买入与主动卖出的净成交额（按分笔成交方向加总），取过去20日累计净主动买入额，除以同期日收益波动率的倒数（即除以每日收益率标准差），得到“主动资金强度”。因子值 = 过去20日主动净流入总和 /（过去20日日收益率标准差 × 过去20日均成交额）。当股票横盘（波动率低）却有持续主动买入时，因子得分高，预期未来5日收益为正。

**期望IC范围**  
- Rank IC 均值约 0.02~0.035（全A除ST、次新）。
- 在震荡市中IC可达0.04以上，单边急跌失效。

**算子约束**  
- 需要分笔数据聚合出 `active_buy_vol` 与 `active_sell_vol`。
- 矩阵计算：`ts_sum(active_buy_vol - active_sell_vol, 20)` / `(ts_std(daily_ret, 20) * ts_mean(volume, 20))`。
- 等价于 1/波动率 × 累计净主动占比，可复用 `ts_scale` 处理分母。

### 2. 分钟级主力/散户背离因子：大单滞销强度

**核心思想**  
价格下跌但主力（>100万元/笔）资金逆势净流入，次日反弹概率显著上升。

**公式逻辑**  
将分钟线按单笔成交额划分为大单（>100万）与小单（<5万），计算过去60分钟内大单净买入额累计值，以及同期分钟级收益率。构造两项的标准化差值：
> `背离得分 = zscore(大单累计净买入) - zscore(累计收益率)`  

当得分 > 1.0 时，预示反转。实际因子可使用连续值：`(大单累计净买入 - 均线) / 大单标准差 - (收益 - 收益均线) / 收益标准差`，并乘以-1（因为价格下跌、大单流入产生正向预测）。

**期望IC范围**  
- 日频Rank IC约 0.015~0.03，盘中因子更强，隔夜衰减快。
- 结合大盘环境后IC可提升至0.035。

**算子约束**  
- 需要分钟级别 `amount` 字段，并按金额流分档：`big_order = amount > 100w`。
- 窗口滚动 `ts_sum(big_order_net, 60)` 计算。
- 算子形式：`(ts_zscore(big_net, 60) - ts_zscore(return, 60)) * -1`。
- 等价于“资金-价格”分位差，需支持 `rolling_apply` + 自定义zscore。

### 3. A股尾盘异动因子：极端尾量振幅比

**核心思想**  
尾盘（14:30-15:00）成交量占全天比例异常放大，且尾盘价格上涨，预示次日资金接力高开的概率较高。

**公式逻辑**  
先定义尾盘特征：
> `尾部量比 = 最后30分钟成交额 / 当日成交额`  
> `尾盘收益 = (收盘价 - 14:30价格)/14:30价格`  

因子值 = `(尾部量比 - 过去20日均尾部量比) / 过去20日尾部量比标准差 × 尾盘收益`。  
即只当量比异常放大且价格同向拉升时给出正向暴露，若量比放大但尾盘下跌，因子值为负。

**期望IC范围**  
- 次日开盘收益的Rank IC约 0.02~0.04，但隔夜反转下衰减快，T+1日全天IC约0.015~0.025。
- 小盘股效果优于大盘股。

**算子约束**  
- 时间切片：需要 `bar_time` 过滤 `14:30` 之后的数据。
- 计算 `tail_volume_ratio`， `tail_return`。
- 算子模版：`ts_zscore(tail_volume_ratio, 20) * tail_return`。
- 此因子可分解为 `zscore` 与 `product` 两个无状态算子组合。
