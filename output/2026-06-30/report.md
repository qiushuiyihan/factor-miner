### 1. WorldQuant Alpha 中的资金流构造逻辑

**核心思想**  
将大单资金流的方向持续性转化为对股价的短期动量预测。

**公式逻辑**  
计算分钟级主动买入与主动卖出的成交额差值，按大单（>20万元）累加得到大单净主动成交额（`BigNetFlow`）。构造 Alpha：  
`alpha = rank(ts_sum(BigNetFlow, 20) / turnover_ma_20)`  
即过去20日大单累计净主动成交额占比（占20日日均换手率），然后横截面排序。逻辑：主力持续净买入的个股，未来5日收益正向。

**期望 IC 范围**  
A股中此因子日频调仓的 Rank IC 约在 **0.02 ~ 0.04** 之间，分层单调性存在但衰减快（半衰期约3-5天）。

**可翻译的算子约束**  
- 模板：`ts_sum( (buy_volume_big - sell_volume_big), d ) / ts_mean(volume, d)`  
- 算子集限制：`ts_sum`, `ts_mean`, `rank`, 整数 `d` 在 `5~60` 之间。
- 过滤：必须使用大单标识（`big_order_tag`），禁用价格直接项。

### 2. 分钟级资金流（主力/散户/大单）的有效组合方式

**核心思想**  
主力与散户行为的背离（主力净买、散户净卖）是更纯净的看涨信号。

**公式逻辑**  
利用分钟成交单分类：单笔成交额>50万标记为主力单，<5万为散户单，其余为中单。分别计算过去N分钟的主力累计净买入（`MajorNet`）和散户累计净卖出（`RetailNetSell`= -散户净买入）。组合因子：  
`SmartMoneyDivergence = rank(MajorNet) + rank(RetailNetSell) - rank(总净流)`  
高值表示主力买入但散户卖出且总净流不高（筹码集中）。用于T+0或次日开盘预测。

**期望 IC 范围**  
在30分钟粒度的日内回转策略中，对次日开盘收益的 IC 约 **0.03 ~ 0.06**；在日频 alpha 中略有下降（0.02 ~ 0.04）。A股T+1限制下，更适用于预测次日开盘跳空。

**可翻译的算子约束**  
- 需要大单/小单标签字段 `order_size_type`（1:主力, 2:中单, 3:散户）  
- 算子：`group_sum`（按类型分组求和）、`ts_sum`、`rank`  
- 序列长度约束：分钟级窗口 `5分钟 ~ 120分钟`。

### 3. A股特有的资金流模式：跌停板效应、北向联动、尾盘异动

**核心思想**  
跌停板上的非散单吃货、北向资金的延时联动、以及尾盘偷袭量都是A股独特Alpha来源。

**公式逻辑**  
- **跌停板吸货因子**：  
  计算当日触及跌停但在跌停板上成交的**主动买入大单金额**除以该股过去5日日均成交额。  
  `DipBuyPower = sum(touched_limit_down * active_buy_big_amount) / avg_turnover_5d`  
  若该比值高，表明主力和聪明钱在跌停板吃货，后续反弹概率高（隔日或2日反转）。

- **北向联动因子**：  
  计算沪深港通北向资金净买卖金额占比的变动率与次日收益的关系。构建：  
  `NorthFlowAccel = (net_inflow_pct_today - net_inflow_pct_yesterday) / volatility_20d`  
  加速流入的个股次日有超额。

- **尾盘异动因子**：  
  计算最后5分钟成交额占全天成交额比例的异常程度（减去20日均值再除以标准差）。  
  `TailVolZ = (volume_last5min / volume_daily - mean_20d) / std_20d`  
  显著升高且股价向上，预示短期动量。

**期望 IC 范围**  
- 跌停板吸货因子 IC 较高（约 **0.04 ~ 0.07**），但样本少、受市场环境干扰大。  
- 北向联动因子 IC **0.02 ~ 0.05**，大盘股中更显著。  
- 尾盘异动因子 IC **0.02 ~ 0.04**，波动性大。

**可翻译的算子约束**  
- 跌停板：需要 `limit_down_tag`, `active_buy_big_amount`, `ts_mean`。  
- 北向：需要 `northbound_net_inflow_pct`, `ts_delta`, `ts_std`。  
- 尾盘：需要 `volume` 的高频切片（识别最后5分钟），`ts_mean`, `ts_std`。

### 4. gplearn 符号回归在因子挖掘中的应用经验

**核心思想**  
在预设的物理意义算子空间内，让遗传规划自动组合资金流多个维度（量、额、大单占比、加速度），发现非线性交互。

**公式逻辑**  
以实际挖掘出的一个有效因子为例（解析后）：  
`alpha = rank( ts_corr( ts_rank(big_net_flow, 10), ts_rank(mid_net_flow, 10), 5 ) ) * sign( ts_delta(big_net_flow, 1) )`  
翻译：若过去10日大单净流排序与中单净流排序的5日相关性为正，且近期大单净流增加，则做多；反之做空。逻辑捕捉大小单同向加速中的主力意愿真实性。 

**期望 IC 范围**  
gplearn挖掘出的复合资金流因子在A股日频调仓的 Rank IC 一般在 **0.03 ~ 0.06** 之间，优于单一资金流因子，但过拟合风险大，需严格样本外验证。一般期望夏普在0.5-1.0（多空组合）。

**可翻译的算子约束**  
- 强制使用算子池：`rank`, `ts_rank`, `ts_corr`, `ts_delta`, `sign`, `+`, `-`, `*`, 保护除法（/改为`ts_scale`或受约束）。  
- 叶子节点限制为：`big_net_flow`, `mid_net_flow`, `small_net_flow`, `total_net_flow`, `turnover`, `volatility` 等资金流/市场指标。  
- 复杂度控制：表达式树深度 ≤ 5，节点数 ≤ 20，交叉验证中IC衰减<20%才保留。
